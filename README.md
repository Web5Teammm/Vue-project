# 影视视界 - 前端项目
一个基于 Vue 3 开发的影视信息展示与交互平台，支持影视浏览、收藏、用户登录/注册、搜索等核心功能，具备响应式布局与流畅的交互体验。

## 项目特性
### 核心功能
- 🎬 影视信息展示：完整展示海报、片名、类型、导演、主演、评分、剧情等全量信息
- ❤️ 收藏功能：支持影视收藏/取消收藏，状态实时同步
- 🔍 搜索功能：关键词模糊搜索，防抖优化减少无效请求
- 👤 用户体系：登录/注册、个人中心、退出登录，状态全局联动
- 📱 响应式布局：适配大屏/小屏设备，保证多终端浏览体验

## 项目运行步骤
### 前置条件
- 已安装 Node.js和 npm
- 已克隆本仓库至本地

### 具体操作步骤
###1. 安装依赖
```bash
# 进入项目目录
cd 项目文件夹名称

# 使用 npm 安装
npm install

### 2. 连接后端数据库
# 在项目根目录下执行，进入 server 文件夹并启动服务
cd server
node server.js

成功标识：终端输出「数据库连接成功」字样
项目已默认配置后端接口地址为 http://localhost:3000，无需手动修改

### 3. 启动前端开发服务器
```bash
# 回到项目根目录；npm 启动
npm run dev

启动成功后，终端会输出本地访问地址（通常为 `http://localhost:5173/`），打开浏览器访问即可。

## 注意事项
1.启动前端前必须先启动后端服务，否则登录、收藏、搜索等依赖接口的功能会失效；
2.依赖安装失败时，优先尝试清除 npm 缓存：npm cache clean --force，再重新执行 npm install；
3.若遇到端口占用问题：
 后端端口（3000）：修改 server/server.js 中的端口配置；
 前端端口（5173）：在 package.json 中调整 dev 脚本，或参考项目构建工具（如 Vite）的端口配置文档；
4.首次运行若数据库连接失败，请检查本地配置。
